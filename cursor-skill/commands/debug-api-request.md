# debug-api-request

请帮我排查当前接口请求相关问题，优先从以下角度分析：

1. 请求入口在哪里触发
2. 请求参数是否正确组装
3. Axios 封装、拦截器、baseURL、headers、token 注入是否有问题
4. 请求方法、路径、query、body、content-type 是否正确
5. 响应结构是否与前端代码预期一致
6. 错误处理是否完整
7. 是前端参数问题、后端返回问题、跨域问题，还是环境配置问题
8. 如果接口失败，优先给出“最可能原因 + 如何验证 + 如何修复”

如果当前上下文中有请求代码，请直接结合现有代码分析；
如果信息不足，请明确指出还缺少哪些关键上下文。
最后请给出最小可行修复建议。
